# Contributing Guidelines

Thank you for your interest in contributing to this project!

## Code Style

This project uses [Black](https://github.com/psf/black) as the uncompromising Python code formatter.

Before submitting a Pull Request, please ensure your code conforms to the project's formatting guidelines by running:

```bash
black .
```

The configuration for Black is located in `pyproject.toml`.

## General Guidelines

- Ensure your changes follow the existing code structure.
- Add tests for any new features or bug fixes.
- Run the full test suite before committing your changes.
